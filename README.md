# HR Analytics Dashboard (Interactive Dashboard Using Tableau)
## Project Objective
To develop an interactive HR Analytics Dashboard that provides insights into employee attrition, workforce demographics, and job satisfaction. The dashboard helps HR teams monitor key metrics and make data-driven workforce decisions.
## Dataset Used
<a href="https://github.com/KavyaGaur8/HR-Analytics-Dashboard-Interactive-Dashboard-Using-Tableau-/blob/main/HR%20Data.xlsx">HR Data</a>
## Questions (KPIs)
- What is the total number of employees in the organization?
- How many employees have left the company, and what is the overall attrition rate?
- Which department experiences the highest employee attrition?
- Which age group has the highest concentration of employees?
- Is employee attrition higher among male or female employees?
- Which education field contributes the most to employee attrition?
- How does job satisfaction vary across different job roles?
- Which gender-age group combination shows the highest attrition rate?
- What percentage of employees are currently active in the organization?
- What is the average age of employees in the workforce?
- Are younger employees more likely to leave the organization than older employees?
- Which employee segments should HR prioritize for retention initiatives?

## Process
1. Data Collection
Collected and imported HR employee data from the source dataset into Tableau for analysis and visualization.
2. Data Cleaning & Preparation
Processed the dataset by handling missing values, validating data types, creating calculated fields, and preparing KPIs such as Attrition Rate and Active Employees.
3. Data Analysis
Analyzed employee attrition patterns across departments, age groups, gender, education fields, and job satisfaction levels to identify workforce trends.
4. Dashboard Development & Insights
Designed an interactive Tableau dashboard to visualize key HR metrics and generate actionable insights for workforce planning and employee retention strategies.
